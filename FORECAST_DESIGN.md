# GBP/CNY 换汇时机预测系统 · 设计文档

> 核心立场：**日频汇率方向几乎不可预测（接近随机游走）**。
> 所以本系统不输出"明天涨还是跌"的点位预测，而是输出对决策真正有用的
> **概率与区间**：未来 N 天内跌到目标价的概率、预期价格区间、最坏情况。
> 任何模型都必须在样本外 **打得过随机游走基准** 才会被采用，否则就用基准。

---

## 1. 要回答的问题（决策导向，而非炫技）

| 问题 | 输出 | 给你的决策意义 |
|---|---|---|
| 未来 7/14/30 天内跌到 ≤9.03 的概率 | P(hit) = 62% | 概率高 → 值得等 |
| 这段时间预期最低能到多少 | E[min] = 9.01 (80% 区间 8.95~9.06) | 看潜在甜头 |
| 如果一直等，price 的扇形区间 | 1/3/7/14/30 天的 80%/95% 带 | 量化等的风险 |
| 现在 vs 等，哪个期望更省钱 | 建议 + 理由 | 一句话结论 |

---

## 2. 数据层

- **主数据**：frankfurter.dev（ECB 日频，免费、跨域）——拉 **3~5 年**历史做训练，而非只 30 天。
- **实时**：Yahoo `GBPCNY=X`（本地监测器已用）做当日最新点。
- **外生变量（关键）**：GBP/CNY ≈ **GBP/USD × USD/CNY**。分解后两条腿各自更"可建模"：
  - GBP/USD：受英国央行利率/英国数据驱动
  - USD/CNY：受中国央行中间价/美元指数 DXY 驱动
  - 一并拉 GBP/USD、USD/CNY、DXY 作为特征。
- **特征工程**：log 收益率、滞后 1/2/3/5/10 日、MA5/10/20、波动率(20日)、RSI、
  星期几（周末效应）、距近 N 日高低点的位置、外生变量的同期收益。

---

## 3. 模型层（分层 + 集成，复杂度递增）

### Tier 0 — 基准（必须存在，所有模型的及格线）
- **随机游走 + 漂移**：明天 = 今天 +（近期平均日漂移）。
- 这是 benchmark；样本外打不过它的模型一律弃用。

### Tier 1 — 统计模型（方向 + 趋势）
- **ETS / Holt 线性趋势**：捕捉短期动量。
- **ARIMA**（对 log 收益率）：自相关结构。
- 期望：相对基准只有**边际**提升，诚实标注。

### Tier 2 — 波动率模型（真正可预测的部分）⭐
- **GARCH(1,1)** 拟合 log 收益率 → 预测未来每日**波动率**。
- 波动率有"聚集性"（大波动后接大波动），这是**确实可预测**的，
  决定了预测区间的宽度——这才是本系统价值核心。

### Tier 3 — 机器学习（捕捉非线性/外生关系）
- **LightGBM / XGBoost**：用 §2 特征做多步预测（含 GBP/USD、USD/CNY、DXY）。
- 谨慎用 LSTM：日频数据样本量小（千级），深网络极易过拟合，**默认不上**，
  除非 Tier 3 GBM 已证明显著超基准再考虑。
- 数据量小（几千行），**本地 Mac 秒级训练，无需 GPU/HPC**。

### Tier 4 — 蒙特卡洛模拟（把上面拼成"概率"）⭐⭐
- 用 Tier 2 的波动率 + Tier 1/3 的漂移，**自助重采样残差**模拟未来 1~30 天
  的 **5000 条价格路径**。
- 从路径分布直接读出：
  - `P(任意一天 ≤ 目标价)`
  - `E[期间最低]` 及其分位区间
  - 各 horizon 的扇形分位带（10/25/50/75/90%）
- 这是输出给你的最终产品，**天然带不确定性**，不骗你。

---

## 4. 评估（防止自欺的关键）

- **Walk-forward 回测**：滚动训练窗，逐日向前预测，绝不用未来数据。
- **对比基准**：RMSE / MAE / **方向准确率**，全部 vs 随机游走。
  - 若方向准确率 ≈ 50%，**如实显示"方向不可预测"**，不粉饰。
- **区间校准（最重要）**：80% 预测区间是否真覆盖 ~80% 的实际值（PIT / coverage）。
  - 校准好 = 概率可信；校准差 = 重估波动率模型。
- **Pinball loss**：评估分位数预测质量。

## 5. 服务与集成

```
GitHub Actions (每天 cron)
   └─ 拉数据 → 训练/更新模型 → 蒙特卡洛 → 写 predictions.json
       └─ git commit & push
            └─ GitHub Pages 自动重建 → 网页读 predictions.json 渲染
```

- **无需你电脑常开**：Actions 在云端跑，纯免费额度内。
- 网页新增模块：
  - 未来扇形区间图（中位线 + 80%/95% 带 + 目标价红线）
  - 概率卡片："7 天内到 9.03 的概率 62%"
  - 一句话建议："波动率偏高，2 周内触及概率不低，建议挂单等"
- **本地监测器**也读 `predictions.json`，在手表提醒里附一句"模型估 X 天内触及概率 Y%"。

---

## 6. 落地路线（建议分两期）

**Phase 1（高价值、诚实核心，建议先做）**
1. 数据：frankfurter 5 年历史 + Yahoo 当日
2. 模型：随机游走基准 + GARCH 波动率 + 蒙特卡洛
3. 输出：P(hit target)、E[min]、扇形区间 → `predictions.json`
4. GitHub Actions 每日跑 + 网页扇形图/概率卡
5. Walk-forward 回测报告（证明区间校准良好）

**Phase 2（锦上添花，先回测再决定上不上）**
6. 加外生变量（GBP/USD、USD/CNY、DXY）+ LightGBM
7. 仅当样本外**显著超随机游走基准**才并入集成；否则保留 Phase 1。

---

## 7. 诚实声明

汇率方向在短期内基本不可预测。本系统的价值在于**量化不确定性**
（你等下去能省多少、风险多大、触及目标的概率），而非承诺低点。
不构成投资建议。
