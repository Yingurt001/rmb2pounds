# rmb2pounds 💷

人民币 → 英镑 换汇时机网页版。一眼看清现在换英镑划不划算。

## 是什么

- 实时显示 **GBP/CNY** 汇率（越低，人民币换英镑越划算）
- **划算度 0–100**：现价在近 30 天区间里离低点多近 + 趋势
- 可设 **目标价**（默认 9.03），跌到目标价以下提示「该换了」
- 近 30 天走势图 + 目标价红线
- 纯静态单页，零后端，浏览器直接跑

## 数据源

[frankfurter.dev](https://frankfurter.dev)（欧洲央行 ECB 参考汇率，工作日更新、免费、支持跨域）。
网页适合「速览趋势 / 看离目标价多远」。

> ⚡ **实时盯盘 + Apple Watch 即时提醒**由本地监测器 `~/fx-monitor/` 负责
> （每 10 分钟查 Yahoo 实时盘，北京 7:00–23:00 跌破目标价就震手表）。

## 本地预览

```bash
# 直接双击 index.html，或起个本地服务器：
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 部署（任选其一）

- **Vercel**：导入此 repo，框架选 "Other"，无需构建，直接发布。
- **GitHub Pages**：Settings → Pages → Branch 选 `main` / `root`。

## 说明

汇率无法预测。本工具只做「逢低买」的数据信号，不构成投资建议。
